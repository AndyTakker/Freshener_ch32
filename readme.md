# Freshener

Берем стандартный освежитель воздуха, который пшикает каждые 36/18/9 минут.
Делаем из него освежитель, который пшикает в зависимости от того, сколько времени горит свет в туалете. 
Чем дольше сидишь в туалете, тем большее число раз после выключения света пшикнет освежитель.
Этот вариант - доработаная версия софта и схемы под использование ch32v003.

Отличия от версии на Attiny85 (https://github.com/AndyTakker/Freshener):

- вместо датчика освещенности используется наличие напряжения от источника питания 5v
- управление моторчиком на основе TA6586.

Остальная логика сохранена.
